# BoolParameter

## Description

`bool` parameter editable in the inspector.

## Arguments
| Type         | Name   | Description                                                                                  | Default         |
|--------------|--------|----------------------------------------------------------------------------------------------|-----------------|
| VariableName | `Name` | Name of the argument exported in the editor, and by which it can be accessed in [GaeaGraph]. | "BoolParameter" |

## Outputs

### Value [bool]

Returns the value as set in [GaeaGraph].
