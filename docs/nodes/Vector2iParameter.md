# Vector2iParameter

## Description

`Vector2i` parameter editable in the inspector.

## Arguments
| Type                                                           | Name (Display Name) | Description                                                                                | Default             |
|----------------------------------------------------------------|---------------|--------------------------------------------------------------------------------------------|---------------------|
| [VariableName](../the-basics/anatomy-of-a-graph.md#slot-types) | `name` (Name) | Name of the argument exported in the editor, and by which it can be accessed in GaeaGraph. | "Vector2iParameter" |

## Outputs

### [Vector2i](../the-basics/anatomy-of-a-graph.md#slot-types) - `value` (Value)

Returns the value as set in GaeaGraph.
