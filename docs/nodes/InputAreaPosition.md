# AreaPosition

## Description

Outputs the position of the area being currently generated.

## Enums

### Input

The available inputs, information about generation accessible from this node.

- World Size
- Area Size
- Area Position
- Area End

## Outputs

### [Vector3](../the-basics/anatomy-of-a-graph.md#slot-types) - `value` (Value)

Returns the selected value, in this case: the position of the area being currently generated.
