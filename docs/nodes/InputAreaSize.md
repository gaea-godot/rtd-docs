# AreaSize

## Description

Outputs the size of the area being currently generated.

## Enums

### Input

The available inputs, information about generation accessible from this node.

- World Size
- Area Size
- Area Position
- Area End

## Outputs

### Value [Vector3]

Returns the selected value, in this case: the size of the area being currently generated.
